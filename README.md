<br>
» Emote selector expanded ☑<br>
   29 to 320 ‼<br>
<br>
» No auto-close on select ☑<br>
   Safe from RSI! ⚠<br>
<br>
» Easily access emotes ✎<br>
<br>
» Client side ☮<br>
<br>
» Configurable ♬<br>
